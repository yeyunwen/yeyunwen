# Switch Profile README Style

This repo includes three profile README variants. Only `README.md` is displayed on your GitHub profile.

## Available Styles

| File | Style | Best for |
| :--- | :--- | :--- |
| `README.md` | **Animated** (default) | Stats cards, typing effect, contribution graph |
| `README.minimal.md` | **Minimal** | Fast loading, clean badges, project table |
| `README.creative.md` | **Creative** | ASCII banner, toolbox, collapsible projects, snake animation |

## How to Switch

```bash
# Switch to Minimal
cp README.minimal.md README.md

# Switch to Animated
cp README.animated.md README.md

# Switch to Creative (includes contribution snake — run workflow first)
cp README.creative.md README.md

# Commit and push
git add README.md
git commit -m "Switch profile README to <style> style"
git push
```

## Contribution Snake (Creative only)

The snake animation is generated by [`.github/workflows/snake.yml`](.github/workflows/snake.yml).

1. Switch to `README.creative.md` as shown above
2. Go to **Actions** → **Generate Snake** → **Run workflow**
3. Wait for the `output` branch to be created, then refresh your profile

The snake image URL: `https://raw.githubusercontent.com/yeyunwen/yeyunwen/output/dist/github-contribution-grid-snake.svg`
